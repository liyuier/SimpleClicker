一个简单的鼠标连点器

代码完全来自 [通义千问](https://www.qianwen.com/)

## 可配置项

* 点击间隔
    
    单位 ms

* 应用的键

    鼠标左键 or 右键

* 启停热键
    
    支持组合热键

## 使用指南

### 设置启停热键

点击设置，按下想要设置的启停热键，点击确认即可

### 使用

* 按下启停热键，鼠标自动开始连点，松开热键不会停止连点

* 再次按下启停热键，鼠标停止连点
